# ReactJS-MeetUp-Application

web Application you can look for MeetUp's create a MeetUp or add MeetUps to favorites.

## Instructions to Run the project

- Install dependencies

```
npm install
```

- Run project locally

```
npm run dev
```

- Build the project

```
npm run build and serve locally
npm install -g serve
serve -s build
```
